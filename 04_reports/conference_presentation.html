üî¨ Methodology

Event Study Framework

Abnormal Returns

$$AR_{i,t} = R_{i,t} - E[R_{i,t} | X_t]$$

Cumulative Abnormal Returns

$$CAR_{i,[t_1,t_2]} = \sum_{t=t_1}^{t_2} AR_{i,t}$$

üìä Factor Models

  • Market model
  • Fama-French 3-factor
  • Carhart 4-factor (main)
  • Fama-French 5-factor

üèõÔ∏è Policy Interaction Framework

Policy Moderation Model

$$CAR_{i,t} = \alpha + \beta_1 ESG\_Change_{i,t} + \beta_2 Policy_{s,t} + \beta_3 (ESG\_Change \times Policy) + \gamma X_{i,t-1} + \epsilon_{i,t}$$

State Policy Index Construction

$$Policy_{s,t} = \frac{1}{4}(Disclosure + Carbon + Renewable + Investment)$$
